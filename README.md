## 🎯 核心特性

- **📦 本地存储优先** — 所有数据保存在浏览器 localStorage，隐私安全
- **📌 小组件固定** — 可将常用工具固定到 Dashboard 实时监控面板
- **🔄 12小时自动缓存** — API 数据自动缓存，防止频繁请求导致封禁
- **🖱️ 一键刷新** — 每个数据小组件支持手动强制刷新
- **🖥️ 独立弹窗** — 工具以 iframe 形式全屏打开，完全独立的运行环境
- **📱 响应式布局** — 自适应桌面端与移动端

## 🧪 家庭资产韧性仿真引擎 v2.0

### 新特性
- **单/双收入模式** — 一键切换家庭收入模式
- **简化/复杂模式** — 支持不同复杂度的模型参数
- **可配置仿真次数** — 500/1000/2000/5000 次可选
- **配置导出导入** — JSON 格式保存/加载场景配置
- **实时进度条** — 仿真运行时显示进度百分比
- **错误处理** — 完善的错误提示和恢复机制

### 核心算法
- **蒙特卡洛仿真** — 每次仿真运行多条路径，覆盖各种市场情景
- **中国个税模型** — 七级超额累进税率，真实反映税后收入
- **分阶段房价路径** — 早期（1-5年）→ 恢复期（6-10年）→ 成熟期（11年+）
- **相关性随机收益** — 金融资产与房产收益率之间相关性建模
- **黑天鹅尾部风险** — 危机概率、失业冲击、资产回撤年度模拟

### 使用场景
- 买房还是租房更适合你？
- 双收入 vs 单收入家庭财务韧性差异
- 极端市场情景下家庭资产抗风险能力
- 不同LPR利率对还贷周期的影响
