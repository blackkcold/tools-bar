# 🧰 Tools Bar (个人工具聚合平台)

[![Demo](https://img.shields.io/badge/Live%20Demo-在线体验-success?style=for-the-badge&logo=githubpages)](https://blackkcold.github.io/tools-bar/)
[![Tech Stack](https://img.shields.io/badge/Tech-HTML5%20%7C%20CSS3%20%7C%20Vanilla%20JS-blue?style=for-the-badge)]()

> 一个**本地优先、纯前端**的个人工具聚合平台。无需后端服务器，支持离线缓存。涵盖计算器、房贷车贷、汇率、存款利率、股市行情、天气预报以及专业的蒙特卡洛家庭资产仿真模型。

---

## ✨ 核心特性

- **📦 本地存储优先**：所有数据保存在浏览器 `localStorage`，确保数据隐私安全。
- **📌 面板可定制化**：支持将常用工具以小组件形式固定到 Dashboard 实时监控面板。
- **🔄 防封禁防抖动**：API 数据自动提供 12 小时本地缓存，防止频繁请求导致 IP 封禁，并支持手动强制刷新。
- **🖥️ 独立沙箱运行**：各类工具以 `iframe` 形式全屏打开，相互之间完全隔离，保证运行环境的独立与稳定。
- **📱 响应式自适应**：完美适配桌面端大屏与移动端小屏。

## 👨‍👩‍👧 重点功能：家庭资产韧性仿真引擎 v2.0

本项目特别内置了基于**蒙特卡洛法则**的家庭资产推演模型，帮助用户进行长达 30 年的财务韧性测试。

### 🌟 亮点特性
- **双模态切换**：一键切换单身/双收入家庭模式；提供极简小白模式与硬核高阶参数模式。
- **高性能仿真**：支持自定义推演次数（500/1000/2000/5000次），并配备实时进度反馈。
- **资产配置存取**：支持将当前财务快照配置导出为 JSON 格式保存或分享，并可随时导入。

### ⚙️ 核心算法引擎
- **蒙特卡洛轨迹**：每次运算生成成千上万条随机轨迹，覆盖牛市、熊市、震荡市等各种经济情景。
- **真实的个税与房贷**：严格按照中国大陆七级超额累进税率、最新 LPR 房贷模型进行税后/税前转换计算。
- **分阶段房产折旧与增值**：精细化房产周期——购房初期（1-5年）跌价/滞涨 → 恢复期（6-10年） → 稳定期（11年以上）。
- **相关性建模与黑天鹅**：金融资产与房产收益率的联动建模；引入失业冲击、系统性危机概率、资产剧烈回撤等尾部风险测试。

### 🎯 解决的问题
1. **买房 vs 租房**，在特定城市 30 年维度下哪个更划算？
2. 极端经济危机下，家庭现金流会断裂吗？
3. 调整 LPR 利率对漫长还贷周期的实际购买力影响有多大？
